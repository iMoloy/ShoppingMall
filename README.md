## 📖 Overview

**Shopping Mall** is a responsive e-commerce product listing website with a clean card-based UI — built with HTML, CSS, and Font Awesome icons. It showcases a multi-category product layout with a shopping cart icon in the navbar.

> **Live at** → [https://imoloy.github.io/ShoppingMall](https://imoloy.github.io/ShoppingMall)

---

## 🛠️ Technologies Used

| Technology | Purpose |
|---|---|
| **HTML5** | Semantic page structure |
| **CSS3** | Custom layout and product card styling |
| **Font Awesome** | Shopping cart, icons |

---

## ✨ Core Features

- **Product Grid** — Responsive card layout for product listings
- **Navbar with Cart Icon** — Top navigation with Font Awesome shopping cart
- **Category Organization** — Products grouped by category
- **Responsive Design** — Adapts to all screen sizes
- **Clean Minimal UI** — Focus on product presentation

## 🚀 Run Locally

1. **Clone the repository**

   ```bash
   git clone https://github.com/iMoloy/ShoppingMall.git
   cd ShoppingMall
   ```

2. **Open in browser**

   ```bash
   open index.html
   ```
